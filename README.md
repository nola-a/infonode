# Infonode - GrpcServer
### Run grpc server
```bash
$ cargo run --bin infonode-server ethbtc # all pairs are supported
```

### Run grpc client (debugging purpose)
note that precisions must be applied to get the right prices 
```bash
$ cargo run --bin infonode-client
```

### Format code
```bash
$ cargo fmt
```

### Running test
```bash
$ cargo test
```
